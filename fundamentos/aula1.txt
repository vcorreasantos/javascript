Fundamentos do JS

JS não é Java

O nome da Linguagem JavaScript é Ecma (EcmaScript)
É possível executar o Js diretamente no navegador da internet

O JavaScript pode ser utilizado também fora do navegador de internet com uso 
da plataforma node.

Particularidades da Linguagem (sintaxe básica)
1° No JavaScript não usamos o conceito de linha e sim de sentença
2° (;) é opcional
3° Podemos usar "" ou '' (não pode misturar) - DICA ⚠️: Usar "" para textos e '' para importações
4° No Js só existe 3 tipo de variaveis (string, number e boolean)
string (caracteres)
number (numeros inteiros ou não inteiros)
boolean (verdadeiro ou falso)
5° No JS as variaveis são dinamicamente tipadas, ou seja, o tipo é determinado quando o valor é atribuido
